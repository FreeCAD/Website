First Bugfix Program Payouts Done

October 2, 2025 Aleksandr Prokudin (169)Announcement (94)FPA (44)Bounty (4)83 words1 min

We’ve just done the first payouts in the pilot bugfix rewards program launched in August. Two contributors have claimed the reward so far: tetektoza fixed 15 confirmed issues, and matthiasdanner fixed another five. Congrats!

We have more money to spend on this, so if you already qualify for a reward or just thinking about joining, please see this blog post for details about the program. Now that we are fully in bugfix mode for v1.1, it’s as good time as any to contribute!
