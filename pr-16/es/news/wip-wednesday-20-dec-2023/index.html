WIP Wednesday - 20 Dec 2023

20 de diciembre de 2023 Aleksandr Prokudin (169)Update (158)WIP (153)188 palabras1 min

This week in FreeCAD development:

  • @Roy-043 continued his work on the new ‘get_param’ function to help having all defaults in one place to avoid inconsistencies.

  • @kpemartin added initial support for reading colors from DXF files.

  • @Syres916 and @WandererFan made small fixes and improvements in the TechDraw workbench. In particular, an underline can now be used instead of the small green squares to identify editable text in templates.

  • @FlachyJoe added the arc-to-arc/circle distance constraint in Sketcher, and @abdullahtahiriyo extended it to arc/circle-to-arc/circle.

  • @paddle’s pull request to add combined rotate / polar pattern tool to Sketcher was merged.

  • @realthunder fixed the bug where the group visibility toggle wouldn’t work on pressing spacebar.

  • @FEA-eng added support for stress tensor components in FEM stress linearization.

PR stats: In the week from Wednesday, 13 Dec to Wednesday, 20 Dec there were 25 pull requests merged, with a total codebase change of +4,745 / -3,361 (not counting translation updates). We had an average merged PR size of +198 / ‑140. 17 new pull requests were opened.

Issue stats: we closed 24 issues, and 13 new issues were opened. Overall, 1,099 issues are currently open.
